El Modal no se abre solo: el boton cambia un estado y React vuelve a renderizar la interfaz.

React Native · Modal & Bottom Sheet 14

Practica guiada · 40 minutos

Animacion con Animated.Value

El truco esta en mover el panel en el eje Y: cerrado abajo, abierto en cero.

const translateY = useRef(new Animated.Value(height)).current;

Animated.timing(translateY, {
  toValue: visible ? 0 : height,
  duration: visible ? 250 : 200,
  useNativeDriver: true,
}).start();

React Native · Modal & Bottom Sheet 21

Practica guiada · 40 minutos

Paso 5: BottomSheet educativo

El componente recibe props y renderiza contenido flexible con children.

function BottomSheet({ visible, onClose, title, height = 320, children }) {
  return (
    <Modal visible={visible} transparent animationType="none">
      <Animated.View style={{ height, transform: [{ translateY }] }}>
        {children}
      </Animated.View>
    </Modal>
  );
}
